## 系统快捷键

|操作|快捷键|
|---|---|
|窗口最大化、最小化（当前程序）|Control + ⌘ + f|
|关闭当前程序|⌘ + q|

## 谷歌浏览器
|操作|快捷键|
|-----|-----|
|打开新的标签页，并跳转到该标签页|⌘ + t|
|关闭当前的标签页|⌘ + w|
|跳转到下一个打开的标签页|⌘ + Option + 向右箭头键|
|跳转到上一个打开的标签页|⌘ + Option + 向左箭头键|
|跳转到特定标签页|⌘ + 1 到 ⌘ + 8|
|跳转到最后一个标签页|⌘ + 9|
|---|---|
|打开“设置”页|⌘ + ,|
|打开“历史记录”页|⌘ + y|
|打开“开发者工具”|⌘ + Option + i|
